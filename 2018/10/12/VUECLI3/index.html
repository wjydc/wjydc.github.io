vue cli3 项目构建

去年做过基础篇的vue 小demo, 由于中间比较忙，就放下了，前不久，再次打开vue 官方文档，进入 vuecli , 发现已经更新到3 了，有点恐慌，才几个月没关注，就迭代了，不得感叹，技术人员必须紧跟信息的跟新，紧随技术的发展，否则就要被市场淘汰了。 于是，又在空闲时间，用新的 vuecli3 搭建起老项目了….
话不多说，看干货，（适合vue初学者，资深前端工程师可绕路）

1 使用 cli 脚手架创建项目

1 全局安装脚手架 npm install -g @vue/cli

2 创建项目 vue create my-project （为自己的项目名称）

3 进入项目 cd my-project
